# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-19

### Added
- Initial release of the portfolio website
- Modern dark theme design inspired by Cursor
- Interactive VS Code editor simulation with:
  - File explorer with expandable folders
  - Multiple file tabs
  - Syntax-highlighted code preview
  - AI chat panel demo
- Fully responsive design for all devices
- Dynamic content system via `data.js`
- Neon/glow effects on logo and branding
- Docker support with nginx
- Sections included:
  - Hero with VS Code demo
  - Trusted By (companies)
  - Feature cards with metrics
  - Experience timeline
  - Testimonials
  - Projects showcase
  - Skills with progress bars
  - Contact information
- Smooth scroll navigation
- Mobile-friendly hamburger menu
- Scroll-triggered animations
- Parallax effects

### Technical Features
- Pure HTML, CSS, JavaScript (no frameworks)
- CSS custom properties for theming
- ES6+ JavaScript
- Multi-platform Docker image (ARM/AMD64)
